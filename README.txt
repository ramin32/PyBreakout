PyBreakout - A python implementation of the classic breakout game

Execute
-------
    To run simply execute 'python game.py' from the command line.

Keys
----
    Mouse and arrow keys move the paddle.
    Esc and q keys quits the game.

Dependencies
------------
    python
    pygame
